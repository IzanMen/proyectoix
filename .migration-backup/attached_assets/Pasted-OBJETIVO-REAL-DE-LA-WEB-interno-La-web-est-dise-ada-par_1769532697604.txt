OBJETIVO REAL DE LA WEB (interno)

La web está diseñada para impresionar a pequeños negocios cuando se les enseña, principalmente desde el móvil, con el fin de que confíen en Izan & Xaloc y quieran contratar el servicio.

No es una web informativa.
Es una prueba de nivel.

El mensaje implícito es:

“Si su propia web es así, imagina la que pueden hacer para mi negocio.”

OBJETIVO PERCIBIDO (externo)

De cara al visitante, la web existe para:

Mostrar profesionalidad extrema

Mostrar modernidad

Mostrar dominio tecnológico

Mostrar cuidado por los detalles

Mostrar que el móvil no es secundario, es protagonista

Nunca se prometen resultados comerciales.

QUÉ DEBE TRANSMITIR (OBLIGATORIO)

SÚPER profesionalidad
No “jóvenes que hacen webs”, sino:

“Gente que juega en otra liga”

Modernidad extrema
Nada clásico, nada genérico, nada plantilla.

Dominio tecnológico
Animaciones fluidas, interacciones cuidadas, sensación premium.

Mobile-first REAL
La web debe verse mejor en móvil que en muchas webs se ven en desktop.

Control y criterio
Todo parece pensado, medido y hecho con intención.

ANIMACIONES (MENSAJE CLAVE PARA REPLIT)

Las animaciones NO son decoración.

Especialmente en móvil.

La web debe dar la sensación de:

Fluidez total

Transiciones elegantes

Nada brusco, nada amateur

“Aquí hay mucho trabajo detrás.”
